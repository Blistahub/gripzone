# RIPZONE - Grip Sport Ranking Platform

Plataforma web para la gestión de rankings y perfiles de atletas de fuerza de agarre (Grip Sport).

## Estado del Proyecto (WIP)
Actualmente es un **MVP (Producto Mínimo Viable)** funcional.
- **Frontend:** HTML5, CSS3 (Animaciones avanzadas), Vanilla JS.
- **Backend:** Node.js + Express.
- **Persistencia:** Sistema de archivos JSON (Prototipado).

## Roadmap (Próximas mejoras)
Estoy trabajando activamente en migrar este prototipo a una arquitectura de producción:
- [ ] Migración de JSON a Base de Datos (MongoDB/SQL).
- [ ] Hashing de contraseñas (bcrypt) y seguridad.
- [ ] Validación de inputs y manejo de sesiones seguro.

## Instalación
1. Clonar el repositorio.
2. `npm install`
3. `node server.js`
4. Acceder a `localhost:3000`
